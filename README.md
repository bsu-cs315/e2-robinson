# CS315 Exercise 2.1

An exercise for CS315 that demonstrates Kinetic motion. This is done with a CharacterBody2D in a platformer game.

## How to play

Press "Play" to start the game.

A: Run left

D: Run right

Space: Jump

Esc: Quit game
